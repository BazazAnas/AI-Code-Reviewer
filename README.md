### 🤖 AI Code Reviewer (Powered by Gemini API)

An intelligent AI-powered code review system that leverages Google Gemini API to automatically analyze source code and generate structured feedback on:

🐞 Bugs & logical errors

🔐 Security vulnerabilities

⚡ Performance bottlenecks

🧹 Code quality & best practices

💡 Optimization suggestions

This project demonstrates real-world LLM integration into a full-stack application.
